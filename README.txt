GoldConstruction — Version Premium (technique)
=================================================

Déploiement GitHub Pages
------------------------
1) Ouvrez le dépôt GitHub : gold-premium
2) Uploadez tous les fichiers de ce ZIP à la racine du dépôt (ou dans /docs si vous publiez depuis docs).
3) Settings → Pages → Build and deployment
   - Source : Deploy from branch
   - Branch : main
   - Folder : /(root)  (ou /docs si vous avez mis les fichiers dans /docs)
4) L’URL : https://claudevieira77580-stack.github.io/gold-premium/

- index.html

- script.js

Notes
-----
- Les liens internes sont adaptatifs grâce au script embarqué : ils fonctionnent même si le dépôt change de nom.
- L’arrière-plan est une image "villa + piscine" avec position bas centrée (pool visible).
- Préchargement minimal "GC" et effets "fade-in" déjà actifs.
